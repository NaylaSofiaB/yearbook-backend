# Documentação da API


GET /alunos

Retorna uma lista de todos os alunos


GET /alunos/:id

Retorna um aluno específico


POST /mensagens

Cria uma mensagem no mural


PUT /alunos/:id

Atualiza o perfil de um aluno específico


DELETE /alunos/:id

Remove o perfil de um aluno


DELETE /mensagens/:id

Remove uma mensagem
